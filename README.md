# SimpleRAT
<h3> RemoteAccessTool </h3>

<h2> Features </h2>

  ~ Download file
  
  ~ Upload file
  
  ~ Screenshot
  
  ~ Handle mutiples client
  
  ~ Sysinfo (only on linux)
  
  <h2> How to use it </h2>
  
  <h1> Server </h1>
  
    ~ Edit line 15,16 to set local IP and listening port
    
<h1> Client </h1>
 
    ~ Edit line 19,20 to set remote IP and remote port
